üöÄ Installation Instructions

Option 1: Quick Start (Recommended)

# Clone the repository
git clone https://github.com/DevCloudesia/RPG-game.git

# Navigate to the folder
cd RPG-game

# Install dependencies
pip install -r requirements.txt

# Run the game
python web_game.py

# Open in browser: http://localhost:8000

Option 2: Download ZIP

  1. Click the "Download ZIP" button above
  2. Extract the ZIP file to a folder
  3. Open a terminal/command prompt in that folder
  4. Run: pip install -r requirements.txt
  5. Run: python web_game.py
  6. Open http://localhost:8000 in your browser

Option 3: Terminal Version

For a classic text-based experience, run:

python main.py

‚ùì Troubleshooting

Python not found?

Download Python from python.org and make sure to check "Add Python to PATH" during installation.

pip not working?

Try using pip3 instead of pip, or python3 -m pip.

Port 8000 in use?

Edit web_game.py and change the port number at the bottom of the file.
